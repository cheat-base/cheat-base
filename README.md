> La plus grande base de données AI de codes de triche pour tous les jeux vidéo.

---

## 🚀 Mise en ligne en 4 étapes (15 minutes)

### Étape 1 — Créer un compte GitHub (gratuit)
1. Va sur **https://github.com**
2. Clique **Sign up**
3. Crée ton compte avec ton email

### Étape 2 — Uploader les fichiers sur GitHub
1. Sur GitHub, clique le bouton **+** en haut à droite → **New repository**
2. Nom du repo : `cheat-base`
3. Laisse tout par défaut → clique **Create repository**
4. Clique **uploading an existing file**
5. Glisse-dépose les 3 fichiers : `index.html`, `vercel.json`, `README.md`
6. Clique **Commit changes**

### Étape 3 — Créer un compte Vercel (gratuit)
1. Va sur **https://vercel.com**
2. Clique **Sign up** → choisis **Continue with GitHub**
3. Autorise Vercel à accéder à GitHub

### Étape 4 — Déployer le site
1. Sur Vercel, clique **Add New Project**
2. Trouve ton repo `cheat-base` → clique **Import**
3. Laisse tout par défaut → clique **Deploy**
4. ✅ Ton site est en ligne ! Tu reçois une URL du type `cheat-base.vercel.app`

---

## 🌐 Connecter ton domaine cheat-base.com

1. Dans Vercel → ton projet → **Settings** → **Domains**
2. Clique **Add Domain** → tape `cheat-base.com`
3. Vercel te donne 2 enregistrements DNS à configurer
4. Va sur **Namecheap** (là où tu as acheté le domaine) → **Domain List** → **Manage** → **Advanced DNS**
5. Ajoute les enregistrements que Vercel t'a donnés
6. Attends 5–30 minutes → ton site est accessible sur **cheat-base.com** ✅

---

## 📊 Google Search Console (pour le SEO)

1. Va sur **https://search.google.com/search-console**
2. Clique **Add property** → tape `https://www.cheat-base.com`
3. Choisis la méthode HTML tag → copie le code
4. Colle le code dans `index.html` entre `<head>` et `</head>`
5. Re-déploie sur Vercel → valide sur Search Console
6. Va dans **Sitemaps** → ajoute `sitemap.xml`

---

## 💰 Activer Google AdSense

1. Va sur **https://adsense.google.com**
2. Clique **Get started** → entre l'URL `https://www.cheat-base.com`
3. Colle le code AdSense dans `index.html` (dans `<head>`)
4. Attends l'approbation Google (1–14 jours)
5. Les pubs s'affichent automatiquement sur tout le site
